About me

A small introduction about myself

Hi, I am Damian and one of my passion is drawing which is a part of my life. I am also very interested in computer graphics 2D (Gimp, Inkscape) and 3D (Blender, VUE) as well. Recently I started learning programming web applications on the frontend side and I improve my IT skills every day. My goal is to became a full stack developer soon.

About my works

I have loved drawing since I was a kid and each done work makes me a better draftsman. I did about 200 drawings till now. Among my works is a lot of characters from my favorite movies, for example Transformers or The Universe of Marvel.

To do much realistic pics I began creating 3D models in Blender including animations. One of them you can see at the top of this page as a background. Others my best models are in Portfolio sections. Recently I started creating cartoon scenes as a files of svg in Inkscape for CoSphere ‘e-learning platform’. These scenes will be used to learn foreign languages. Up to now I did about 30 scenes you can see in portfolio section as well.

Currently I am HTML, CSS and JavaScript learner. From this trio I am the best familiar with HTML and CSS. I can create styles using SCSS preprocessor. Up to now I did a lot of responsiveness mock templates using my own layout grid written in SCSS and Bootstrap and Zurb Foundation frameworks as well. I’m also WordPress and AngularJS learner.

I am still learning JavaScript language. I have written about hundred small functions including a few unit tests for each. I am learning programming using Test Driven Development approach to make my code better. For unit tests I use Jasmine framework. I wrote a small ToDo app in vanilla javascript as well using Model-View-Controller pattern.

I really like jQuery library. It is much simpler to me to make my website dynamic than using vanilla js. I work with Git version control system. For each part of code I need to implement I create a new branch and then open a pull request to merge it into master or rc (release candidate) branch.

All my frontend work I did till now you can see on my Bitbucket and Github accounts.
